Just add flex classes into any parent element and it's child element will be flexed easily

flex classes

Flex Classes are shorthanded like this

These are 12 basic flexbox ShortHand classes. Later We wil make much interesting thing out of these 12 classes.
You can learn how to implement these classes here

ShortHand Classes CSS flexbox Propery
.flex-center { display: flex; justify-content: center; }
.flex-left { display: flex; justify-content: flex-start; }
.flex-right { display: flex; justify-content: flex-end; }
.flex-bottom { display: flex; align-items: flex-end; }
.flex-middle { display: flex; align-items: center; }
.flex-top { display: flex; align-items: flex-start; }
.flex-space-around { display: flex; justify-content: space-around; }
.flex-space-between { display: flex; justify-content: space-between; }
.flex-stretch { display: flex; align-items: stretch; }
.flex-col { display: flex; flex-direction: column; }
.flex-row { display: flex; flex-direction: row; }
.flex-wrap { display: flex; flex-wrap: wrap; }

How it works

Implementing flexbox ShortHand classes

Let's take a parent element containing a child element, like this 


<div class="parent"> 
    <div class="child">
        <p>child</p>
    </div>
</div>

You can make something like that after styling parent and child element(I have not added flexbox-shorthand classes yet)
Result :

child

Now we are going to add those 12 basic flexbox-shorthand classes

.flex-center

add "flex-center" class to parent 


<div class="parent flex-right"> 
    <div class="child">
        <p>child</p>
    </div>
</div>

See result :

child


.flex-right

add "flex-right" class to parent 


<div class="parent flex-center"> 
        <div class="child">
            <p>child</p>
        </div>
</div>

See result :

child


.flex-left

add "flex-left" class to parent 


<div class="parent flex-left"> 
    <div class="child">
        <p>child</p>
    </div>
</div>

See result :

child


.flex-bottom

add "flex-bottom" class to parent 


<div class="parent flex-bottom"> 
    <div class="child">
        <p>child</p>
    </div>
</div>

See result :

child


.flex-middle

add "flex-middle" class to parent 


<div class="parent flex-middle"> 
    <div class="child">
        <p>child</p>
    </div>
</div>

See result :

child


.flex-top

add "flex-top" class to parent 


<div class="parent flex-top"> 
    <div class="child">
        <p>child</p>
    </div>
</div>

See result : It looks same like flex-left because default horizontal position is left and vertical is top here. you will see we can change both horizontal and verical positions using flexbox and we can put child element at 9 positions within the same parent element. See here
